# 《影梦》(Silhouette Dream) 地点设定

> 简化版本 - 核心场景与功能

## 一、主要场景

### 1.1 废弃小诊所（医生据点）
- **功能**：治疗恢复、信息获取、物品交易
- **氛围**：安静专业，略带消毒水气味，光线昏暗温暖
- **视觉**：褪色蓝白招牌，生锈铁门，整洁内部，玻璃药瓶
- **叙事**：理性、秩序、治疗象征，梦境中的稳定据点
- **游戏作用**：
  - BOSS"医生"的居所和工作场所
  - 恢复生命值，解除负面状态
  - 提供梦境世界知识和线索

### 1.2 破旧图书馆（休息据点）
- **功能**：休息存档、知识收集、完全安全区域
- **氛围**：静谧安全舒适，"旧而不破"的感觉
- **视觉**：爬山虎外墙，古典建筑，深色木门，明亮电灯
- **叙事**：知识、记忆、文明的象征，被守护的秩序
- **游戏作用**：
  - 完全安全的避风港
  - 自动保存游戏进度
  - 通过旧书获取线索和知识

### 1.3 小区公园（负面回忆场所）
- **功能**：情绪挑战、记忆触发、社交恐惧考验
- **氛围**：空旷冷漠，略带威胁感，阴沉天空
- **视觉**：大象滑梯，紫藤花长廊，白色混凝土框架，生锈器材
- **叙事**：孤独和社交恐惧的具象化，童年创伤载体
- **游戏作用**：
  - 生成基于孤独和恐惧的敌人
  - 触发小男孩回忆片段
  - 帮助其他孤独NPC的支线任务

### 1.4 小男孩的家（故事核心）
- **功能**：故事推进、情感连接、个人物品互动
- **氛围**：温馨表象下的孤独现实
- **视觉**：简单晚餐永远摆放，时钟停在特定时间
- **叙事**：家庭温暖与内心孤独的对比
- **游戏作用**：
  - 可互动的个人物品（日记、画作）
  - 展现现实生活与梦境连接的桥梁
  - 关键剧情触发点

## 二、废弃场景

### 2.1 废弃商场
- **功能**：探索收集、环境解谜
- **特征**：玻璃天幕裂缝处植物生长，胶囊电梯内有涂鸦
- **叙事**：曾是家庭购物场所，现在空无一人

### 2.2 废弃学校
- **功能**：故事推进、物品收集
- **特征**：教室黑板上保留最后一节课内容
- **叙事**：小男孩曾经就读的学校

### 2.3 废弃游乐场
- **功能**：技能挑战、隐藏探索
- **特征**：旋转木马上动物有缺失的眼睛
- **叙事**：曾是快乐场所，现在荒凉寂寥

## 三、场景功能关联

| 场景 | 主要功能 | 安全等级 | 情感基调 |
|------|----------|----------|----------|
| 小诊所 | 治疗恢复 | 安全 | 理性、秩序 |
| 图书馆 | 休息存档 | 完全安全 | 静谧、安全 |
| 小区公园 | 情绪挑战 | 危险 | 孤独、恐惧 |
| 小男孩的家 | 故事核心 | 安全 | 温馨、孤独 |
| 废弃商场 | 探索收集 | 中等 | 荒凉、神秘 |
| 废弃学校 | 故事推进 | 中等 | 怀旧、伤感 |
| 废弃游乐场 | 技能挑战 | 高危险 | 诡异、不安 |

## 四、设计原则

### 4.1 情感氛围优先
- 每个场景有明确的情感基调
- 视觉元素服务于情感表达
- 游戏机制与场景氛围一致

### 4.2 旧而不破美学
- 陈旧但不破烂，有人维护的痕迹
- 时间流逝的美感，而非破败感

### 4.3 功能性明确
- 每个场景必须有明确的游戏功能
- 功能与叙事紧密结合
- 玩家能直观理解场景用途

### 4.4 多层次叙事
- **表层**：视觉和环境叙事
- **中层**：互动和机制叙事
- **深层**：隐喻和象征叙事

## 五、视觉主题

### 5.1 颜色调性
- **蓝色调**：小诊所（医疗蓝）、图书馆（书本蓝灰）
- **绿色调**：图书馆（爬山虎绿）、小区公园（植物绿）
- **棕色调**：废弃场景（铁锈棕、木色棕）
- **灰白色**：建筑框架（混凝土灰白）

### 5.2 材质质感
- **陈旧材质**：生锈金属、褪色油漆、磨损木材
- **反光元素**：玻璃药瓶、水洼反射、金属光泽
- **自然元素**：爬山虎、杂草、藤蔓、苔藓
