# 《影梦》(Silhouette Dream) 角色设定

> 简化版本 - 核心角色与命名规范

## 一、主角

### 1.1 小男孩（现实身份）
- **角色ID**：`CHAR_BOY_001`
- **核心概念**：爱幻想的小男孩，影子宿主
- **关键特质**：
  - 敏感、孤独，渴望情感连接
  - 患有慢性健康问题，需要定期看医生
  - 白天的经历影响夜晚的梦境内容
- **游戏作用**：
  - 现实成长载体：白天选择永久解锁新能力
  - 情绪源头：情绪投射到梦境生成怪物
  - 叙事视角：展现故事的现实层面

### 1.2 影子（梦境身份）
- **角色ID**：`CHAR_SHADOW_001`
- **核心概念**：依靠光存在的共生种族，小男孩的影子
- **存在特性**：
  - 白天寄生在小男孩身上
  - 夜晚进入梦境世界自由活动
  - 黑暗中将逐渐失去自我意识
- **背景故事**：
  - 远古太阳衰败后幸存的影子
  - 被魔女的泪水唤醒获得独立意识
  - 在梦境中探索以维持存在
- **游戏作用**：玩家在梦境中的可操控角色

## 二、主要NPC

### 2.1 医生
- **角色ID**：`NPC_DOCTOR_001`
- **核心概念**：梦境免疫系统管理员
- **多重身份**：
  - **表象**：灾厄散布者，敌方高级干部
  - **里象**：秘密盟友，治疗者
  - **现实映射**：现实中帮助小男孩的医生
- **象征意义**：必要的痛苦、治疗性创伤、成长必经之痛
- **外貌特征**：
  - 身高约190cm，瘦削威严
  - 沾污白色长袍，瘟疫医生鸟嘴面具
  - 面具下是温和面容，工具发出治愈金光
- **游戏功能**：
  - 诊所据点：治疗恢复、物品交易
  - BOSS战：挑战玩家恐惧
  - 剧情转折：揭示真相的盟友

### 2.2 小熊（泰迪熊）
- **角色ID**：`NPC_BEAR_001`
- **核心概念**：寻找心脏的泰迪熊
- **背景故事**：
  - 小男孩床头的泰迪熊
  - 承载小男孩对情感连接的渴望
  - 寻找"心脏"实为小男孩自我接纳的投射
- **外貌特征**：
  - 破旧棕色泰迪熊
  - 右眼纽扣脱落，有缝补痕迹
  - 胸口有空心洞（象征缺失）
- **象征意义**：
  - 情感投射：渴望从外部获得安慰
  - 自我欺骗：以为心脏在别处
  - 自我接纳：意识到心跳就在自己心中

### 2.3 魔女
- **角色ID**：`NPC_WITCH_001`
- **核心概念**：幽灵引导者，傲娇的同伴
- **真实身份**：梦主（小男孩）妈妈的投射
- **性格特质**：
  - 端庄优雅，傲娇，隐藏柔软内心
  - 只有夜晚出现，阳光照射会透明消逝
  - 本体是戒指/玩偶/耳环，不能远离
- **核心动机**：
  - **表面谎言**：摧毁太阳，拯救影子
  - **真实渴望**：找到生前家园，了解父母是否爱她
- **剧情弧线**：
  - 引导者 → 真相揭露 → 冲突决裂 → 理解告别
- **象征意义**：与母亲的和解，隐藏的情感需求

## 三、命名规范（简化）

### 3.1 格式规则
```
[类型]_[角色名]_[序号]
```

### 3.2 类型定义
| 类型代码 | 全称 | 说明 |
|---------|------|------|
| `CHAR` | Character | 玩家操控角色 |
| `NPC` | Non-Player Character | 非玩家角色 |
| `BOSS` | Boss | BOSS级敌人（预留） |
| `MOB` | Mob | 普通怪物（预留） |

### 3.3 角色名缩写
| 缩写 | 对应角色 | 说明 |
|-----|---------|------|
| `BOY` | 小男孩 | 现实主角 |
| `SHADOW` | 影子 | 梦境主角 |
| `BEAR` | 小熊 | 泰迪熊NPC |
| `DOCTOR` | 医生 | 医生NPC |
| `WITCH` | 魔女 | 魔女NPC |

### 3.4 当前角色ID列表
| 角色ID | 角色名 | 类型 | 状态 |
|--------|--------|------|------|
| `CHAR_BOY_001` | 小男孩 | 主角 | 已实现 |
| `CHAR_SHADOW_001` | 影子 | 主角 | 已实现 |
| `NPC_DOCTOR_001` | 医生 | NPC | 已实现 |
| `NPC_BEAR_001` | 小熊 | NPC | 已实现 |
| `NPC_WITCH_001` | 魔女 | NPC | 规划中 |

### 3.5 序号规则
- **格式**：3位数字，从 `001` 开始
- **用途**：区分同一角色的不同形态或版本
- **示例**：
  - `CHAR_SHADOW_001` = 影子基础形态
  - `CHAR_SHADOW_002` = 影子觉醒形态（预留）
  - `NPC_WITCH_001` = 魔女幽灵形态
  - `NPC_WITCH_002` = 魔女本体形态（预留）

## 四、角色关系

### 4.1 核心关系网
- **小男孩 ↔ 影子**：寄生共生，现实与梦境的双重身份
- **影子 ↔ 魔女**：唤醒者与被唤醒者，引导者与追随者
- **影子 ↔ 医生**：恐惧对象与治疗者，敌对与盟友
- **影子 ↔ 小熊**：情感投射者与投射对象，帮助与自我发现
- **小男孩 ↔ 医生**：现实中的医患关系，信任与恐惧

### 4.2 情感主题
- **自我认同**：影子寻找存在意义
- **情感连接**：建立与接纳各种关系
- **恐惧面对**：转化恐惧为成长动力
- **自我欺骗**：意识到外部寻求的答案就在内心
