# 《影梦》(Silhouette Dream) - 一个剧情向横版解谜冒险游戏

> **一句话讲清楚这个项目**：一款让玩家扮演小男孩的影子，在梦境与现实之间穿梭，通过解谜、探索与轻度紧张互动体验剧情与情感的横版冒险游戏。

### 世界观概述

- **双重世界**：表层是充满象征意义的梦境冒险世界；底层真相是爱幻想小男孩的梦，影子是共生种族
- **现实-梦境联动**：小男孩入睡后影子进入梦境探索，怪物是现实经历/情绪的投射
- **影子种族设定**：依靠光而存在的古老种族。远古时期太阳永不落下，影子一族繁荣昌盛；时代迁移，太阳衰败导致黑夜诞生，影子一族落入黑夜便会逐渐失去自我、消逝在黑暗中
- **核心主题**：内心成长与自我和解，通过梦境冒险反映现实心理蜕变

### 关键角色

- **小男孩（玩家现实身份）**：爱幻想的小男孩，患有慢性健康问题，敏感、孤独、渴望情感连接。白天的生活经历会影响夜晚的梦境内容
- **影子（玩家梦境身份）**：小男孩的影子，依靠光而存在的共生种族。在黑夜中必须不断探索来维持自我意识，通过梦境冒险成长
- **魔女**：幽灵形态的神秘引导者，原型是梦主（小男孩）的妈妈。端庄优雅却傲娇，隐藏着柔软的内心。她编造谎言引导主角，真实动机是寻找生前的家、了解父母对自己的真实想法
- **寻找心脏的小熊**：破旧的泰迪熊，胸口有空洞。相信找到心脏就能获得完整生命，实际上是小男孩孤独感的投射
- **医生**：梦境中的假反派，现实中真心帮助小男孩的医生。穿着鸟嘴面具，看似带来灾厄，实则是梦境世界的免疫系统管理员
- **狼桃骑士**：来自秘鲁大森林的西红柿骑士，大大咧咧、粗犷憨厚。为了追寻正义、得到认可踏上冒险，最终在弥留之际悟到自我认可的真理，化作西红柿种子

### 核心玩法特色

- **探索解谜导向**：无战斗设计，通过解谜、躲避、安抚等方式与敌人互动
- **睡眠阶段系统**：
  - **清醒期**：选择梦境区域，查看记忆碎片，设置目标
  - **浅睡期**：探索 + 简单解谜 + 躲避巡逻型敌人
  - **深睡期**：紧张躲避 + 恐惧管理 + 资源收集
  - **REM期**：高潮解谜 + 情感共鸣 + 剧情解锁
- **核心循环**：进入梦境 → 探索场景 → 收集记忆 → 解谜/躲避 → 推进剧情
- **玩家操作**：移动(WASD)、互动(E/空格)、快速行走(Shift)、躲藏(Ctrl)
- **敌人互动**：躲避、安抚、解谜、引导（非战斗设计）
- **情绪投射**：怪物和障碍是现实负面情绪（如恐惧、孤独）的具象化，通过互动来面对
- **医学元素设计**：疾病→异常状态，病原→怪物，心理→技能
- **特殊睡眠状态**：梦中梦、清醒梦、鬼压床
- **多结局系统**：普通结局 + 魔女真结局（理解并阻止魔女，在故乡阳光下告别）

## 🔧 技术实现要点

### 引擎与工具

- **主引擎**：Godot 4.x（2D引擎对像素游戏支持优秀）
- **美术工具**：Aseprite / Pixelorama
- **版本控制**：Git + GitHub
- **项目管理**：Trello看板 + 飞书文档

### 关键实现设置

1. **像素美术**：纹理导入Filter → Nearest，关闭Mipmaps，启用像素对齐
2. **场景结构**：节点化设计，分离UI层、世界层、实体层
3. **互动系统**：基类继承模式，支持多种交互类型
4. **数据持久化**：Resource格式保存永久进度
