# scenes/ — 场景文件

Godot `.tscn` 场景，按功能分为三类：

## 入口

| 场景 | 说明 |
|-----|------|
| main.tscn | 主入口，自动加载game_manager |
| main_menu.tscn | 主菜单 |

## 关卡

| 场景 | 说明 |
|-----|------|
| demo_level.tscn | 演示关卡（主要试玩场景） |
| maze_level.tscn / maze_level2.tscn | 迷宫关卡 |
| simple_test.tscn / test_level.tscn | 测试沙盒 |

## Boss战

| 场景 | 说明 |
|-----|------|
| bear_boss_scene.tscn | 小熊Boss |
| doctor_boss_scene.tscn | 医生Boss |
| clinic_scene.tscn | 诊所场景（医生据点） |
| playground_scene.tscn | 公园场景 |
