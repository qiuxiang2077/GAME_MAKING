# 《影梦》(Silhouette Dream) 游戏设定

> 横版剧情向解谜冒险 | Godot 4.6 | 千禧年梦核风

---

## 一、世界观

### 核心概念

| 概念 | 定义 |
|-----|------|
| **影子 (Shadow)** | 依靠光而存在的共生种族，寄生在人类宿主身上。拥有独立的思维和情感 |
| **梦境世界 (Dream World)** | 由梦主（做梦者）的潜意识构建的虚拟空间，物理法则不稳定 |
| **情绪投射 (Emotional Projection)** | 将内在情绪外化为外部实体的机制 |

### 影子存在状态

- **依附态**：白天寄生在宿主身上，跟随宿主的行动
- **游离态**：夜晚进入梦境世界，独立活动
- **消逝态**：在黑暗中停留过久，逐渐失去自我

### 情绪投射表

| 情绪 | 投射表现 | 典型形态 |
|-----|---------|---------|
| 恐惧 | 威胁性实体 | 怪物、陷阱 |
| 孤独 | 虚空、缺失 | 空洞、迷宫 |
| 愤怒 | 破坏性力量 | 狂暴敌人 |
| 悲伤 | 沉重、缓慢 | 沼泽、锁链 |
| 焦虑 | 压迫感 | 追逐、倒计时 |

### 世界起源

- **远古**：永恒太阳照耀，影子一族繁荣发展
- **太阳衰败**：黑夜诞生，大部分影子失去意识，少数学会寄生
- **现代**：寄生成为常态，影子通过梦境进入人类精神世界

## 四、故事主线

### 篇章一：魔女的谎言

魔女编造谎言引导影子摧毁"太阳"，真实动机是寻找生前的家。影子在旅程中发现真相，最终理解魔女的痛苦，陪伴她回到破旧但温馨的家。魔女在故乡的阳光下消逝。

### 篇章二：寻找心脏的小熊

小熊在梦境中寻找"心脏碎片"，相信这样就能获得生命。影子帮助小熊收集碎片，最终发现心跳声其实是小男孩自己的。小熊学会了自我接纳。

### 篇章三：逃离那个医生

医生在梦境中以恐怖形象出现，带来灾厄。影子不断逃避，最终被逼入绝境时发现医生的真实身份——梦境的"免疫系统管理员"。从恐惧到理解，医生成为盟友。

## 五、玩法机制

### 核心循环

```
进入梦境 → 探索场景 → 收集记忆碎片 → 解谜/躲避敌人 → 推进剧情 → 醒来
```

### 玩家操作

| 操作 | 按键 | 说明 |
|-----|------|------|
| 移动 | WASD | 角色基础移动 |
| 奔跑 | Shift | 加速移动，消耗体力 |
| 互动 | E / 空格 | 收集物品、触发机关、对话 |
| 躲藏 | Ctrl | 进入躲藏状态躲避敌人 |

### 睡眠阶段系统

| 阶段 | 梦境层级 | 环境特征 | 核心玩法 |
|-----|---------|---------|---------|
| 清醒期 | 表层梦境 | 现实与梦境的模糊边界 | 选择路线，与NPC对话 |
| 浅睡期 | 过渡层 | 场景不稳定，记忆片段切换 | 环境解谜，躲避危险 |
| 深睡期 | 核心梦境层 | 黑暗、压抑，接近潜意识深处 | 恐惧管理，极限生存 |
| REM期 | 活跃梦境层 | 色彩鲜艳，逻辑荒诞 | 紧张追逐，剧情推进 |

### 敌人系统（非战斗型）

| 类型 | 行为 | 应对方式 |
|-----|------|---------|
| 巡逻型 | 固定路径巡逻 | 观察规律，绕行通过 |
| 追踪型 | 发现玩家后追踪 | 利用地形甩开，躲藏 |
| 潜伏型 | 固定在特定区域 | 寻找安全路径，触发机关 |

### 解谜系统

| 谜题类型 | 说明 |
|---------|------|
| 推箱子 | 推动箱子到目标位置，压制机关或搭建通路 |
| 开关谜题 | 按特定顺序或时机触发多个开关 |
| 记忆锚点 | 找到场景中的记忆碎片，解锁隐藏区域 |
| 情感共鸣 | 利用特定情绪状态改变环境（如恐惧使门打开） |

### 收集系统

- **记忆碎片**：散落在梦境各处，收集推进剧情、解锁能力
- **情绪残留**：击败/避开情绪怪物后获得，用于强化能力

## 六、美术风格

### 风格定位：千禧年梦核风 (Chinese Millennium Dreamcore)

> "你可以回到这里，但这里已经没有人了"

- 梦境：梦幻、超现实、色彩鲜明但略带诡异
- 现实：温暖写实的日常场景
- 核心氛围：怀旧感、孤独感、探索未知

### 色彩规范

| 角色 | 色调 | 说明 |
|-----|------|------|
| 梦境主色调 | 冷蓝 #4A7C9B、青绿 #5B8A72 | 梦幻、神秘 |
| 现实主色调 | 暖黄 #E8D5B7、米白 #F5F0E8 | 温暖、熟悉 |
| 恐惧/危险 | 暗红 #8B2252、深紫 #3D1F3D | 压迫感 |
| 希望/安全 | 暖橙 #E8A87C、淡金 #D4A574 | 安慰感 |

### 像素规格

- 角色尺寸：32×32px
- 场景元素：16×16px 或 32×32px
- 动画帧数：待机4帧，行走8帧
- 文件格式：PNG（透明背景）
