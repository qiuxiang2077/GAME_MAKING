# 《影梦》(Silhouette Dream) - 可玩Demo运行说明

## 📁 项目结构

```
GAME_MAKING/
├── assets/          # 资源文件夹
│   ├── art/         # 美术资源
│   └── audio/       # 音频资源
├── scenes/          # 场景文件
│   ├── main.tscn    # 主场景
│   └── demo_level.tscn # 演示场景
├── scripts/         # 脚本文件
│   ├── player.gd    # 玩家控制
│   ├── sleep_manager.gd # 睡眠阶段系统
│   ├── enemy.gd     # 敌人AI
│   └── collectible.gd # 记忆碎片收集
├── ui/              # UI界面
│   ├── hud.tscn     # 游戏HUD
│   └── hud.gd       # HUD逻辑
├── project.godot    # Godot项目文件
├── README.md        # 项目说明
└── RUNNING.md       # 运行说明
```

## 🚀 如何运行

### 方法一：使用Godot引擎打开
1. 下载并安装 [Godot 4.x 引擎](https://godotengine.org/download)
2. 启动Godot引擎
3. 点击 "Import" 按钮
4. 选择项目文件夹中的 `project.godot` 文件
5. 点击 "Import & Edit" 按钮
6. 在编辑器中按 `F5` 或点击 "Play" 按钮运行游戏

## 🎮 游戏操作

| 操作 | 按键 | 功能 |
|------|------|------|
| 移动 | WASD/方向键 | 上下左右移动 |
| 互动 | E/空格 | 与物体互动 |
| 跑步 | Shift | 加速移动 |
| 躲藏 | Ctrl | 进入/退出躲藏状态 |

## 🌟 游戏特色

- **核心玩法**：探索解谜导向，无战斗设计
- **睡眠阶段系统**：清醒期、浅睡期、深睡期、REM期
- **恐惧值系统**：深睡期会积累恐惧值
- **敌人AI**：简单的巡逻和追逐行为
- **记忆碎片**：收集记忆碎片推进剧情
- **HUD界面**：显示睡眠阶段、恐惧值和剩余时间

## 📝 演示场景内容

在演示场景中，你可以：
- 控制影子角色在场景中移动
- 与红色敌人互动（会触发检测）
- 收集黄色的记忆碎片
- 观察睡眠阶段的变化
- 体验恐惧值的积累
